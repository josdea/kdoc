Welcome to the KDoc theme for Kirby CMS. This theme is free to use, and is on github here.

What is this?

KDoc is starter theme for building documentation websites using Kirby CMS. It's based on DOCter which is used for the same purpose, but based on Jekyll. This version also includes some inspiration from 18F's Guides Template, especially for the expanding menu idea.

This project is maintained by Nate Steiner.
